## Leyenda de Análisis de Riesgos
* **Peligro:** *de 0 (sin peligro) a 3 (no hay línea)*
* **Evacuación** *de 0 (se puede salir en cual quier momento) a 3 (sin salidas / encañonado)* 
* **Suma y Categorización**
    * **0** *Sin riesgo*
    * **1** *Bajo*
    * **2-3** *Medio*
    * **4-5** *Alto*
    * **6** *Crítico*
